## for
### cpp版
```cpp
for(int i; i < 5; i++){
    cout << i << '\n';
}
```
### Python版
```py
for i in range(0, 5, 1):
    print(i)
```
##### 輸出
```py
0
1
2
3
4
```
#### Python特殊方法
```py
n = [1, 2, 3, 4, 5]
for i in n:
    print(i)
```
##### 輸出
```py
1
2
3
4
5
```
